## ✅ AUTENTICACIÓN ENDPOINTS
```bash
# 1. Registrar usuario
curl -X POST https://bossbudgetapi-production-539b.up.railway.app/api/auth/registrar \
  -H "Content-Type: application/json" \
  -d '{"username":"juanperez","nom":"Juan","apell":"Perez","pass":"12345678","email":"juan@ejemplo.com","prof":"Developer"}'

# 2. Iniciar sesión (guardar token manualmente de la respuesta)
curl -X POST https://bossbudgetapi-production-539b.up.railway.app/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"juan@ejemplo.com","pass":"12345678"}'

# 3. Guardar token (reemplaza con el token real)
TOKEN=""

# 4. Obtener perfil
curl -X GET https://bossbudgetapi-production-539b.up.railway.app/api/auth/perfil \
  -H "Authorization: Bearer $TOKEN"

# 5. Actualizar perfil
curl -X PUT https://bossbudgetapi-production-539b.up.railway.app/api/auth/perfil \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"nombres":"Juan Carlos","apellidos":"Perez Gomez","profesion":"Full Stack"}'

# 6. Cambiar contraseña
curl -X PUT https://bossbudgetapi-production-539b.up.railway.app/api/auth/cambiar-password \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"contraseñaActual":"12345678","nuevaContraseña":"nuevoPassword123"}'

# 7. Cerrar sesión
curl -X POST https://bossbudgetapi-production-539b.up.railway.app/api/auth/logout \
  -H "Authorization: Bearer $TOKEN"
```

## PRESUPUESTO ENDPOINTS
```bash
# 1. Crear presupuesto (solo si no tienes uno)
curl -X POST https://bossbudgetapi-production-539b.up.railway.app/api/presupuestos \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"nombre":"Mi Presupuesto","valor":50000,"fecha_inicio":"2025-01-01","fecha_fin":"2025-12-31","categoria":"General"}'

# 2. Ver presupuesto
curl -X GET https://bossbudgetapi-production-539b.up.railway.app/api/presupuestos \
  -H "Authorization: Bearer $TOKEN"

# 3. Ver presupuesto específico (ID 1)
curl -X GET https://bossbudgetapi-production-539b.up.railway.app/api/presupuestos/1 \
  -H "Authorization: Bearer $TOKEN"

# 4. Actualizar presupuesto
curl -X PUT https://bossbudgetapi-production-539b.up.railway.app/api/presupuestos/2 \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Presupuesto 2025",
    "monto": 60000,
    "fecha_inicio": "2025-01-01",
    "fecha_fin": "2025-12-31",
    "categoria": "General"
  }'

# 5. Eliminar presupuesto (cuidado)
curl -X DELETE https://bossbudgetapi-production-539b.up.railway.app/api/presupuestos/1 \
  -H "Authorization: Bearer $TOKEN"
```
